Library

Ezra Natanael

Welcome to my personal collection of books and resources.

If you’re a software engineer, these are must-reads!

  • Designing Data-Intensive Applications

    An excellent book that covers core concepts in distributed systems, databases, and big data processing.

  • Database Internals

    A deep dive into database structures and operations, exploring topics like data organization, log-structured databases, and distributed systems. Perfect for expanding on concepts introduced in the first book.

  • 100 Go Mistakes and How to Avoid Them

    Learning from mistakes is invaluable. This book highlights common errors in Go and has helped me write cleaner, more efficient code.

  • Pro Git

    No software engineer should be without Git knowledge! This book covers it all - and it’s free!

  • Grokking Algorithms

    A beginner-friendly guide to algorithms, showing you how to apply core computer science concepts to practical programming challenges.

  • Efficient Go

    As systems grow more complex and markets demand speed, software efficiency becomes essential. This book emphasizes performance optimization driven by observability, helping engineers create cost-effective and successful products.
