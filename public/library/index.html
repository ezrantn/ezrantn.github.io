Library

Ezra Natanael

Here’s my stash of favorite tech books and resources that I keep coming back to.

These have been game-changers for me - check them out!

  • Clean Architecture: A Craftsman’s Guide to Software Structure and Design

This book is packed with practical tips to make your software easier to build and maintain over the long run. Uncle Bob brings decades of experience to the table, giving you straightforward advice to make the right design choices that can make or break a project.

  • Designing Data-Intensive Applications

This one’s a beast! It breaks down all the complicated stuff about distributed systems and databases in a way that actually makes sense.

  • Database Internals

Ready to go deeper into databases? This book is your friend. It dives into all the nitty-gritty details and builds on the concepts from DDIA.

  • 100 Go Mistakes and How to Avoid Them

Trust me, you’ll want this one if you’re writing Go. I’ve made most of these mistakes myself - save yourself the headache!

  • Pro Git

Look, Git is everywhere in software dev. This book’s got you covered AND it’s free! Can’t beat that.

  • Grokking Algorithms

Scared of algorithms? Don’t be! This book makes them way less intimidating with lots of real-world examples.

  • Efficient Go

Want your Go code to absolutely fly? This book’s all about making things fast and keeping costs down. Super practical stuff!
